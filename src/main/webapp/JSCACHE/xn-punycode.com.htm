Punycode Converter

Punycode is a simple ASCII compatible encoding designed for the use of Internationalized Domain Names in Applications (IDNA). It allows the reversible transformation of a Unicode string to a string of characters (letters, digits, and hyphens), used in hostnames, and back again.
